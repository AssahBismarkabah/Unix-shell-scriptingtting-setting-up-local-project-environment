This project is a Bash project. To run the scripts in this project, you can use the following commands:

With execution permissions:

./script_name.sh


**Without execution permissions:**

bash script_name.sh


For example, to run the script `my_script.sh`, you would run the following command:

./my_script.sh


To make the script executable, you can use the following command:

chmod +x my_script.sh


To make the script non-executable, you can use the following command:

chmod -x my_script.sh
